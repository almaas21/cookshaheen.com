# CookShaheen Theme

A beautiful custom theme for CookShaheen - Your Ultimate Food & Recipe Destination.

## Features

- ✅ Stunning responsive landing page
- ✅ Smooth scroll animations
- ✅ Mobile-friendly navigation
- ✅ Recipe cards with hover effects
- ✅ Newsletter subscription section
- ✅ Testimonials section
- ✅ Beautiful footer with social links

## Installation

1. Upload the `cookshaheen` folder to `/wp-content/themes/`
2. Go to WordPress Admin → Appearance → Themes
3. Activate the "CookShaheen" theme

## Customization

Edit the files in the theme folder to customize:
- `style.css` - Colors, fonts, and layouts
- `page-landing.php` - Landing page content
- `assets/js/main.js` - Animations and interactions
